GT Cars

Our objective is to create a fun game with innovative mechanics and a fresh user experience. We made the decision to develop a game exclusively for computer platforms in light of the present environment, in which more people stay at home and spend a lot of time on their computers.

The players goal in this game is to cross the finish line while gathering coins and keys. However, they must travel over treacherous terrain that is dotted with several hazards designed to slow them down and keep them from finishing the race. Players can unlock different cars to outrun the chasing cops and increase their chances of success by collecting coins.

The project was collaboratively created in Unity using C#.

In this collaborative project my part was the creation and improvement of the UI and UX of the application. I aimed to create a seamless and interesting user interface for users to interact with through a combination of clever layout and intuitive user experiences. I made sure that each aspect of the UI and UX was carefully created in accordance with the project's objectives and user expectations by working closely with the team.
